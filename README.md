### Data Transformation & Analysis Project

## Overview
This project analyzes global access to basic drinking water services between *2000 and 2020*.  
Using spreadsheet-based data analysis techniques, the project transforms raw data into meaningful insights about trends, inequalities, and progress across *national, rural, and urban populations*.

The analysis aligns with *United Nations Sustainable Development Goal 6 (SDG 6): Clean Water and Sanitation*.

---

## Objectives
The main goals of this project were to:

- Understand the structure of a real-world global dataset  
- Investigate how water access data is distributed across years  
- Calculate *Annual Rates of Change (ARC)* for water access  
- Compare access across national, rural, and urban populations  
- Identify countries with *full access* to basic water services  
- Analyze differences in progress between rural and urban areas  
- Compare water access improvements across global regions  
- Present findings using charts, histograms, and pivot tables

---

## Key Insights
- Water access data was not collected annually for all countries, requiring careful year-difference calculations.  
- Rural areas often showed higher rates of improvement, suggesting targeted development efforts.  
- Countries with 100% access impact average ARC values and must be treated separately.  
- Regional analysis reveals uneven global progress and highlights areas needing intervention.
